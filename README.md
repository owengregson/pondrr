<h2>Overview</h2>
<p>
    <strong>pondrr</strong> is a real-time automated 3D 4x4 tic-tac-toe bot running in Java through HTTP protocol to a web3 frontend.
</p>

<h2>Guide</h2>
<ul>
	<li><strong>Start the server: </strong>javac *.java && java TTT3</li>
</ul>
